# vaadin-tippyjs

An opinionated sample demo for a [Tippy.js](https://atomiks.github.io/tippyjs/) tooltip with HTML in [Vaadin](https://vaadin.com/docs/latest/).

![Screenshot](Screenshot.png)

## Run

```shell
./gradlew build bootRun
```
